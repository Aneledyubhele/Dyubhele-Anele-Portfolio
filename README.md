# Personal Portfolio

A minimal, responsive personal portfolio website built with HTML, CSS, and JavaScript.

## 🚀 Getting Started

1. Open `index.html` in your browser.
2. Update the placeholder text (Your Name, project descriptions, contact links) to match your personal details.

## 🛠️ Customize

- Modify `styles.css` to change colors and layout.
- Add / replace project cards in `index.html`.
- Add images to a new `assets/` folder and reference them from the HTML.

## 📦 Deploy

You can deploy this portfolio easily using services like GitHub Pages, Netlify, or Vercel.

---

> Tip: Rename the repository folder and update the links in `index.html` after you deploy to keep your portfolio working correctly.
